# DayNite Engine

DayNite Engine is a custom 2D RPG engine written in C# on top of MonoGame.

## Goals
- Classic JRPG-style gameplay (turn-based combat, overworld exploration)
- 2D tile-based worlds
- Data-driven systems (JSON)
- Clear separation between engine and game logic

## Tech
- C# / .NET
- MonoGame (DesktopGL)
- VS Code on Windows
